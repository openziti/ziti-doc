<h3>In OpenZiti You Will Find</h3>

* A zero trust [overlay mesh network](../../reference/glossary.mdx#network-overlay-overlay) composed of a controller, edge routers, and fabric routers
* Lightweight [tunnelers](../../how-to-guides/tunnelers/index.mdx) that add zero trust to existing applications with no code changes
* [SDKs](../../reference/developer/sdk/index.mdx) for embedding zero trust directly into your application for the strongest security posture
* Zero trust principles such as [authorize before connect](../../learn/core-concepts/security/authentication/00-auth.md),
[continual authorization](../../learn/core-concepts/security/authorization/00-auth.md),
[least-privilege access](../../learn/core-concepts/security/authorization/policies/overview.mdx)

OpenZiti gives you **zero trust, high performance networking on any Internet connection**, without VPNs. Add it to existing applications with tunnelers, or embed it directly with our SDKs for the strongest posture.

:running_woman: Ready to deploy your first network? Jump right in, feet first, and follow along with one of our
[quickstart guides](../get-started/quickstarts.md)!
